# ITV Kidney Prediction System

A Machine Learning based Kidney Disease Prediction System built using Python, Scikit-learn, and Flask.  
This project predicts whether a patient is likely to have kidney disease based on medical attributes provided by the user.

---

# 📌 Project Overview

The main goal of this project is to develop an end-to-end Machine Learning pipeline for kidney disease prediction and deploy it using a Flask web application.

The system performs:
- Data preprocessing
- Feature engineering
- Model training
- Pipeline creation
- Prediction through Flask UI
